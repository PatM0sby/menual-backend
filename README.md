# Menual App
## Backend

Frontend to be found [here](https://github.com/Sofia-Christ/menual)

Install
```
npm install
```

Run
```
npm start
```

API Localhost:3000

POST /food Request

Header:
```
'Content-Type' : 'application/json'
```
Body (JSON):
```
{"query":"theFoodImLookingFor"}
```

POST /food Response
